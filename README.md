# Rumble-in-Chains
AUTEURS : Raphaël Gonzales aka Eliade
		Jasmin Pradat
		Florentin Rollet

Nom du jeu : Lucha Loca Encadenadas

Installation : dézipper le dossier dans le registre de votre choix et lancez Lucha_Loca_Encadenadas.exe (avec la miniature qu'il faut). C'est tout. Ne pas lancer les autres executables

But du jeu : chaque joueur doit rejoindre le spotlight de sa couleur et se faire prendre en photo en appuyant sur X. Le premier à 25 points gagne

Contrôles :
	Manette de Xbox:
		
		Stick directionnel gauche : se déplacer
		Stick vers le abs quand on est en l'air : tomber plus vite
		A : Sauter / Choisir dans les menu
		B + direction : Dash
		B sans direction : faux dash / Annuler dans les menu / Quitter 
		X : attaquer
		X + direction : d'autres attaques
		Y : tirer l'adversaire vers soit
		Y + Haut : balancer l'adversaire vers le haut
		Y + Bas : le balancer vers le bas
		Gachette arrière : Bouclier

	Clavier:
		Joueur Bleu:
			ZQSD : bouger
			Space : Sauter
			Shift gauche: Dash
			X : attaquer
			B : Tirer
			N : Bouclier
		Joueur Rouge : //Il faut un pad numérique
			OKLM : bouger
			Shift droit : Sauter
			2 : Dash
			4 : Attaquer
			8 : Tirer
			6 : Bouclier
